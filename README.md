> 🚀 AI-powered system to detect oral diseases from images using transfer learning.

---

## 📌 Overview
This project predicts oral diseases such as:
- Oral Cancer  
- Gum Disease (Periodontal Disease)  

Users can upload an image and get:
- Predicted disease  
- Confidence score  
- Precautions  
- Warnings  

---

## 🎯 Features
- 🖼 Image Upload Interface  
- 🤖 Deep Learning Prediction  
- 📊 Confidence Score  
- ⚠️ Health Warnings & Precautions  
- 🌐 Interactive UI (Gradio)  

---

## 🛠 Tech Stack
- Python  
- TensorFlow / Keras  
- OpenCV
